Les statistiques sur les objets trouvés dans les gares SNCF en 2022

Nous avons récupéré les données dans les gares SNCF en 2022 des objets perdus. Nous avons fait plusieurs datavisualisations pour vous montrer ces données intéressantes.

L'évolution des objets perdus en gare dans le temps (par mois) :

Nous pouvons voir que c'est le mois de Mai où il y a le plus d'objets perdus.

Nous avons regroupé les objets perdus dans des catégories d'objets suivantes :

Les 3 types d'objets les plus perdus sont la bagagerie, les appareils électroniques et les porte-monnaies.

Nous avons voulu maintenant quelques sont ces objets les plus perdus :

Les 3 natures d'objets perdus les plus perdus sont les habits chauds qu'on pose en arrivant dans le train, les sacs et les porte-monnaies/portefeuilles. Faites attention à ces objets !
On peut voir quand mettant la nature des objets, aucun objet n'est dans la catégorie des appareils électroniques qu'on avait vu avant.

Essayez maintenant de ne plus oublier vos objets en gare, vérifiez que vous partez avec ! Merci d'avoir lu ce blog et j'espère que vous avez appris des choses.

Points d'amélioration possible des datavisualisations :

  • Retravailler les couleurs des bares des histogrammes. (bleu sur bleu n'est pas très lisible).
  • Regrouper les catégories les moins signifiantes dans une catégorie "autre" pour gagner en lisibilité.
  • Proposer une visualisation par gare des différents oublis / types d'oublis.
  • Proposer les mêmes visualisations sans la catégorie "bagages" qui est de façon évidente le type d'objet le plus oublié.
